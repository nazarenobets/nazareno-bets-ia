# Passo a passo Supabase

1. Acesse https://supabase.com
2. Crie uma conta ou faça login.
3. Clique em New Project.
4. Nome sugerido:
   `nazareno-bets-ia`
5. Crie uma senha forte do banco.
6. Aguarde o projeto abrir.
7. Vá em SQL Editor.
8. Clique em New query.
9. Abra no projeto o arquivo:
   `database/supabase_schema.sql`
10. Copie tudo, cole no SQL Editor e clique em Run.
11. Vá em Project Settings > API.
12. Copie:
   - Project URL
   - service_role key

Esses dados vão nas variáveis do servidor.
