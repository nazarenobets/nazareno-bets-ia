# Nazareno Bets IA Web 3.5 — Ajustes de Produção

Correções desta versão:

- API-Football e Sportmonks agora são reconhecidas quando estão configuradas no servidor Render;
- teste de conexão das APIs passa pelo servidor;
- checklist final marca automaticamente o que já está configurado;
- textos de versão corrigidos para 3.5;
- painel de produção mais claro.

## Como atualizar no Render

1. Envie/substitua estes arquivos no GitHub.
2. Aguarde o Render fazer novo deploy automático ou clique:
   Manual Deploy > Deploy latest commit.
3. Abra a plataforma online e vá em Configurações.
