# Publicar Nazareno Bets IA Web 3.3 no Railway

## 1. Criar conta no Railway
Acesse railway.app e crie sua conta.

## 2. Criar projeto
- New Project
- Deploy from GitHub ou Upload

## 3. Variáveis
Configure:
```env
NODE_ENV=production
APP_DOMAIN=nazarenobets.com
API_FOOTBALL_KEY=sua_chave
SPORTMONKS_TOKEN=seu_token
SUPABASE_URL=sua_url_supabase
SUPABASE_SERVICE_ROLE_KEY=sua_service_role_key
ADMIN_USER=admin
ADMIN_PASS=troque_essa_senha
```

## 4. Domínio
Vá em Settings > Domains e adicione:
```text
nazarenobets.com
```

## 5. Resultado
A plataforma fica online 24h, sem tela preta.
